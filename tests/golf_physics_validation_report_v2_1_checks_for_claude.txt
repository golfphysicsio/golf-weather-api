## 7) Drag/lift equations and coefficient models
This section is a **doc audit** (guide is unchanged):
- Drag: `F_drag = ½ ρ A C_d v_rel²` (standard form).
- Lift: `F_lift = ½ ρ A C_l v_rel²` with direction set by spin axis (Magnus).
- C_d and C_l are simplified linear functions of spin parameter with caps (0.50, 0.40).
Recommendation: Keep these as-is for an app if calibrated, but consider adding a small “aero sanity” table in test data (typical Cd/Cl ranges) so future edits don’t drift.
