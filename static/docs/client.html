Units (Imperial & Metric)

Global Support

All API responses include both imperial and metric units, allowing you to display values in your users' preferred unit system without additional API calls.

The units Parameter

All endpoints accept an optional units query parameter:

Value Description
imperial Default. Indicates preference for Fahrenheit, yards, mph
metric Indicates preference for Celsius, meters, km/h

Note: The units parameter sets units_preference in the response, but all responses include both unit systems regardless of this parameter.

Dual-Unit Response Format

Measurements are returned as objects containing both units: 

{
  "adjusted": {
    "carry": {
      "yards": 220.9,
      "meters": 202.0

      "mph": 10,
      "kmh": 16.1
    },
    "altitude": {
      "feet": 500,
      "meters": 152
    }

  "units_preference": "imperial"
}

API Endpoints

POST /api/v1/trajectory

Calculate adjusted ball trajectory based on shot data and weather conditions.

Request Body

Field Type Required Description
shot.ball_speed_mph number Yes Ball speed in mph (60-225)
shot.launch_angle_deg number Yes Launch angle in degrees (-10 to 60)
shot.spin_rate_rpm number Yes Spin rate in RPM (0-15000)
conditions.temperature_f number Yes Temperature in Fahrenheit (-40 to 130)
conditions.humidity_percent number Yes Relative humidity (0-100)
conditions.altitude_ft number Yes Altitude in feet (0-15000)
conditions.wind_speed_mph number Yes Wind speed in mph (0-100)
conditions.wind_direction_deg number Yes Wind direction (0-360, 0=headwind)

Response

Field Description
adjusted Weather-adjusted trajectory metrics
baseline Trajectory without weather effects
impact_breakdown Individual effect of each weather factor
trajectory_points Array of x,y,z coordinates for visualization

print(f"Adjusted carry: {result['adjusted']['carry_yards']} yards")

FAQ

How do I get an API key?

Email us at golfphysicsio@gmail.com with your company name, expected use case, and estimated request volume. We'll provision an API key and send it to you within 24 hours.

What does wind direction 0 mean?

Wind direction is measured in degrees where 0 = direct headwind, 90 = crosswind from the right, 180 = tailwind, and 270 = crosswind from the left. This is relative to the shot direction.

How accurate are the calculations?

Our physics model is calibrated against TrackMan data and accounts for air density, wind effects, and altitude. Accuracy is typically within 2-3% of measured results under normal conditions.

Can I upgrade my rate limit?

Yes! Contact us to upgrade from Free to Standard or Enterprise tier. Rate limit changes take effect immediately after upgrade.

Is there a sandbox environment?

The production API can be used for development. Free tier keys have a 60 requests/minute limit which is sufficient for development and testing.

Do you support metric units?

Yes! All API responses include both imperial and metric units in every response. You can use the units=metric query parameter to indicate your preference, but the response will always contain both unit systems. This allows you to display values in your users' preferred units without additional API calls.

How do I display distances in meters?

All distance measurements are returned as objects with both units: {"yards": 220.9, "meters": 202.0} Simply access the meters property to display metric values. No conversion needed!
